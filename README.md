# 📚 BookNest — E-Commerce Web App 

BookNest is a modern book e-commerce web project built with **Nuxt 3**.  
It is designed to provide a fast, responsive, and simple.

---

## 🚀 Key Features
- 💳 Online store system powered by **Nuxt 3**
- ⚡ High performance with **Vue 3**
- 🎨 Modern UI using **shadcn-vue**
- 📦 Product management
- 👤 User authentication and login system
- 🛒 Shopping cart and checkout feature
- 🔐 Role-based access control
- ☁️ **Supabase Storage** support

---

## 🧩 Technologies Used
- [Nuxt 3](https://nuxt.com)
- [Vue 3](https://vuejs.org/)
- [Typescript](https://www.typescriptlang.org/)
- [TailwindCSS](https://tailwindcss.com/)
- [shadcn-vue](https://www.shadcn-vue.com/)
- [Supabase](https://supabase.com/)
- [Prisma](https://www.prisma.io/)
- [PostgreSQL](https://www.postgresql.org/)
